Chrome Extension

BizReminder

Ett Chrome-tillägg som påminner dig om skatte- och bolagsdeadlines — med systemnotiser direkt i webbläsaren.

Förhandsgranskning av popup

Funktioner

  • Systemnotiser via chrome.notifications
  • Automatiska alarm med chrome.alarms
  • On/off-toggle med sparat läge
  • Länkar till IRS, Skatteverket & Bolagsverket
  • Manifest V3 — modern extension-standard

Installera lokalt

1
Ladda ner filerna Klona eller ladda ner detta repo
2
Öppna Chrome Gå till chrome://extensions
3
Aktivera Developer mode Växeln uppe till höger
4
Klicka Load unpacked Välj mappen biz-reminder-extension/
